
# ZakatGoldApp (Option C) - Full Project

This project is configured to be compatible with Android Studio Narwhal 2025.1.3 (AGP 8.7 + Gradle 8.8).

Before opening in Android Studio:
- Replace placeholders in `activity_about.xml` and `MainActivity.java` (appUrl) with your details (name, student no, program code, GitHub username).
  Minimum supported Gradle version is 8.9. Current version is 8.8.

Please fix the project's Gradle settings.
Change Gradle version in Gradle wrapper to 8.9 and re-import project
Open Gradle wrapper properties
Gradle Settings.

How to open:
1. Extract the ZIP
2. Open folder in Android Studio (File > Open)
3. Let Android Studio download Gradle 8.8 and sync

Important: If Gradle JDK version problem appears, set Gradle JDK to a JDK >= 17 (File > Settings > Build, Execution, Deployment > Build Tools > Gradle).
